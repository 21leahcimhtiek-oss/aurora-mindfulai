# ZenAI

> Personalized AI meditation and mindfulness

## Features
- Guided meditations
- Mood tracking
- Breathing exercises
- Sleep sounds
- Stress insights

## Stack
- Next.js 14 (App Router) + TypeScript
- Tailwind CSS
- OpenAI GPT-4o-mini
- Stripe Subscriptions
- Vercel deployment

## Quick Start

```bash
npm install
cp .env.example .env.local
# Fill in API keys
npm run dev
```
